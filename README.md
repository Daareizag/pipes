# Nombre

Pipes de Angular.

## Caracteristicas

Imparte conocimiento sobre los pipes de Angular, propios y personalizados.

## Tecnologias

pipesApp, es una aplicacion que imparte conocimiento sobre los pipes propios de angular y los que un desarrollador puede personalizar dependiendo de sus necesidades. Se trabajo de una manera modularizada. Se usaron pipes numericos, ortograficos y personalizados, cada uno de ellos viene con su peque√±a definicion. En cuanto a los pipes personalizados se crearon 3, se encargan de ordenar en una lista de heroes por color, por nombre o por su caracteristica (Vuela o No Vuela). Se usaron estilos y componentes de una web que te los proporciona llamada PrimeNg. La aplicacion se programo con Angular, typescript. El framework de trabajo fue Visual Studio Code.
